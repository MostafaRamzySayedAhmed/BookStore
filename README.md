# Book Store
It is a Responsive Business to Customers (B2C) E-Commerce System, Providing The Basic Features That Both Users & Admins Need

# The Features
* The Responsive, Simple & Classic Design.
* Fully Responsive Dashboard for Admins to Make All CRUD Operations on All of the System Modules
  **(Users, Books, Categories, Sub-Categories & Carts)**.
* The Admin Can Control the Activation of the Users' Accounts.
* The User Can View Books in a Simple & Highly-Organized Manner.
* The Books are Distributed over Sub-Categories.
* The Sub-Categories are Distributed Over Categories.
* The User Can Know Details about their Favorite Books like Author, Publisher, Edition & Number of Pages.
* The User Can Specify the Quantity of the Book he Wants to Buy.
* The User Can Calculate the Price of the Total Quantity.
* The User Can Add Books He Wants to Buy After Calculating it's Total Price to his Cart.
* The User Can View his Profile Info.
* The User Can Edit his Profile Info.
